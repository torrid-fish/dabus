# DaBus
Final project of software studio.

## Frontend branch
This branch is the frontend part of our project.

## How to run?

### Step 0: Install packages
```
npm install
```

### Step 1: Run Metro
```
npm run start
```

### Step 2: Choose which device to use

r - reload the app
d - open developer menu
i - run on iOS
a - run on Android
